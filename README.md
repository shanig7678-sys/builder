# MBM Building Construction
This is the UI update of mbm building construction website.

And website created in React JS using some built-in packages.

## npm Commads
``` bash
npm i
npm start
```

## Package Use
``` bash
npm i react-icons
npm i react-router-dom
npm install swiper
npm install react-slick slick-carousel
```
